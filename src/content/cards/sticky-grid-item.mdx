<Card>
  <div>
    # {frontmatter.name}

    If you want to make a grid item sticky, you need to `align-self: start`.

    Grid items have a default attribute for `align-items` to `stretch`, meaning all children have the same height. you can avoid this by setting `align-items: start;`
  </div>
